# OKO R1 skin kit (source)

- **`r1-skin-kit-offline.html`** — offline bundle with lens shapes (HAL eye, GLaDOS head/potato, TARS monolith) and accent tokens (`amber` / `red` / `blue`).
- Persona markdown lives in **`backend/personas/`** (`persona_hal9000.md`, `persona_glados.md`, `persona_tars.md`).

Android maps skins → lens shape + accent; backend loads the matching persona file per `hello.skin`.
