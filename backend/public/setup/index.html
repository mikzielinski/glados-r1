OKO Integracje

Kreator GitHub & UiPath

Połącz repozytoria i Orchestrator bez ręcznego n8n. OAuth lub token — klikasz, autoryzujesz, gotowe.

  • GitHub — OAuth App lub Personal Access Token
  • UiPath — External Application w Automation Cloud
  • Automatyczna konfiguracja skilli GLaDOS (bez webhooków n8n)

GitHub

Wybierz OAuth (zalecane) albo wklej PAT.

GitHub → Settings → Developer settings → OAuth Apps → New
Callback URL: …

UiPath Orchestrator

External Application w Automation Cloud → Admin → External Applications.

Redirect URI: …
Scopes: OR.Administration OR.Jobs OR.Queues OR.Folders OR.Execution

Podsumowanie

Po zakończeniu GLaDOS używa lokalnych endpointów — n8n nie jest wymagane.
